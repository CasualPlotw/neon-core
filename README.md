# 🌌 NEON-CORE: QUANTUM SCANNER

![Security](https://img.shields.io/badge/Security-Omni--Level-00f2ff?style=for-the-badge)
![Status](https://img.shields.io/badge/System-Active-ff007f?style=for-the-badge)

**NEON-CORE**, siber güvenlik odaklı, neon estetiğiyle tasarlanmış bir şifre analiz ve istihbarat terminalidir. 

## 🛡️ Özellikler
* **Quantum Analysis:** Şifre gücünü gerçek zamanlı skorlar ve kırılma süresini tahmin eder.
* **IP Intelligence:** Kullanıcının canlı IP ve bölge verilerini otomatik olarak çeker.
* **Vocal Feedback:** Önemli güvenlik durumlarında sesli uyarı verir.
* **Neon-Grid UI:** Göz yormayan, modern ve şık siber arayüz.
* **Glitch Alert:** Zayıf şifre girişlerinde sistem "Breach" moduna girer.
